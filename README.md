# Vectors
A python helper lib for vectors
## to import
```python
# import
import Vector.vector as v
```

## to create vector
```python
# create
new_vector = v.Vector(<x value>, <y value>)
```

## to acess functions
```python
# Ex
new_vector.set_mag(10)
```
